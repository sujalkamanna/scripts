### ✅ Other useful **install / setup scripts** you can add to your repo

To make your DevOps repo more complete and industry-ready, consider adding these:

#### 🔐 Security & Access

* `vault.sh` – Install HashiCorp Vault (secrets management)
* `trivy.sh` – Container & filesystem vulnerability scanner
* `falco.sh` – Runtime security for Kubernetes
* `keycloak.sh` – Identity & access management (SSO)

#### 📊 Monitoring & Logging

* `prometheus.sh` – Metrics collection
* `grafana.sh` – Dashboards & visualization
* `loki.sh` – Log aggregation
* `elastic-stack.sh` – ELK stack setup

#### ☁️ Cloud & Kubernetes

* `helm.sh` – Kubernetes package manager
* `kubectl-plugins.sh` – krew + useful plugins
* `argocd.sh` – GitOps continuous delivery
* `kind.sh` – Kubernetes in Docker (alternative to Minikube)
* `eksctl.sh` – EKS cluster management (AWS)

#### 🔁 CI/CD & Dev Tools

* `gitlab-runner.sh` – GitLab CI runner
* `github-actions-runner.sh` – Self-hosted GitHub runner
* `maven.sh` / `gradle.sh` – Build tools
* `nodejs.sh` – Node.js & npm setup

#### 🧱 Infrastructure / OS

* `packer.sh` – Image building
* `awscli.sh` – AWS CLI install
* `azurecli.sh` – Azure CLI install
* `gcloud.sh` – Google Cloud CLI
* `nfs-server.sh` – NFS for shared storage
* `haproxy.sh` – Load balancer
* `nginx.sh` – Reverse proxy / ingress
